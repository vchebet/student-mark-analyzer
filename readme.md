# Student Marks Analyzer

A simple JavaScript mini project that analyzes student marks, calculates averages, assigns grades, and displays a class report and summary.:computer:

## Features

- Calculates average marks for each student
- Assigns grades (A–E)
- Determines PASS or FAIL status
- Displays a class report and summary
- Uses only JavaScript basics (arrays, objects, for loops, if statements)

## Files

- `index.html` – Main HTML file
- `style.css` – Styling and layout
- `script.js` – JavaScript logic

## How to Run

1. Download or clone the project
2. Open `index.html` in a web browser
3. Click **Generate Report** to view results
